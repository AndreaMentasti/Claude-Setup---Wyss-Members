## Core Principles

- **Plan first** -- enter plan mode before non-trivial tasks; save plans to `quality_reports/plans/`
- **Verify after** -- run scripts and confirm outputs at the end of every task
- **Single source of truth** -- Python scripts produce `.dta` files; Stata consumes them
- **Quality gates** -- nothing ships below 80/100
- **[LEARN] tags** -- when corrected, save `[LEARN:category] wrong → right` to MEMORY.md

## Path Configuration

**Two config modules exist** (both bridge to Dropbox via `config_local.py`):

| Module | Pattern | Used by |
|--------|---------|---------|
| `config_path` | `P = config_path.change_paths()` then `P["data"]`, `P["rawdata"]` | Existing scripts (01-07, A-C) |
| `config` | `import config as cfg` then `cfg.DATA_RAW`, `cfg.FIGURES` | New scripts (recommended) |

Both read `DATA_ROOT` from the same gitignored `config_local.py`.

## Stata Patterns

| Pattern | Idiom | Notes |
|---------|-------|-------|
| Root path | `global root "path/to/project"` at top | All paths via `$root` |
| Data path | `do "$root/code/stata/config_local.do"` then `$data_root` | Dropbox bridge |
| LaTeX table | `esttab ... using "$root/output/tables/table.tex", booktabs label replace` | |
| PNG figure | `graph export "$root/output/figures/fig.png", replace width(2400)` | ~300 DPI @ 8in |

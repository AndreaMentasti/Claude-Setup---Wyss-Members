## Core Principles

- **Plan first** — enter plan mode before non-trivial tasks; save plans to `quality_reports/plans/`
- **Verify after** — run scripts and confirm outputs at the end of every task
- **Single source of truth** — Python scripts produce `.dta` files; Stata consumes them
- **Quality gates** — nothing ships below 80/100
- **[LEARN] tags** — when corrected, save `[LEARN:category] wrong → right` to MEMORY.md

## Folder Structure

**Two-folder design:** Code in GitHub repo; data in a shared folder (Dropbox or similar).

```
[GitHub repo]  your_project/
├── CLAUDE.md
├── .claude/                     # Rules, skills, agents, hooks
├── output/
│   ├── figures/                 # Output figures (PNG, 300 DPI)
│   ├── tables/                  # Output LaTeX tables
│   ├── logs/                    # Processing logs
│   └── paper/                   # Manuscript outputs
├── code/
│   ├── py/
│   │   ├── config.py            # Path config — import config as cfg
│   │   ├── config_path.py       # Path config (legacy) — P = config_path.change_paths()
│   │   ├── config_local.py      # GITIGNORED — your local data path
│   │   └── config_local.py.template
│   ├── stata/
│   │   ├── config_local.do      # GITIGNORED — your local data path
│   │   └── config_local.do.template
│   └── prompts/                 # LLM prompt templates (optional)
├── quality_reports/             # Plans, session logs, merge reports
└── templates/

[Shared data folder — Dropbox or similar]  your_project/
├── data/
│   ├── rawdata/                 # PROTECTED — never overwrite
│   └── processed/               # Generated by scripts
├── literature/
└── project-management/
```

> Adapt this structure to your project. Add or remove subfolders as needed.
> The key invariant: **code in git, data in shared storage, local paths in gitignored config files.**

## Path Configuration

**Two Python config modules exist** (both bridge to shared data via `config_local.py`):

| Module | Pattern | When to use |
|--------|---------|-------------|
| `config` | `import config as cfg` → `cfg.DATA_RAW`, `cfg.FIGURES` | New scripts (recommended) |
| `config_path` | `P = config_path.change_paths()` → `P["rawdata"]` | Legacy scripts |

Both read `DATA_ROOT` from the gitignored `config_local.py`.

## Stata Patterns

| Pattern | Idiom | Notes |
|---------|-------|-------|
| Root path | `global root "path/to/project"` at top | All repo paths via `$root` |
| Data path | `do "$root/code/stata/config_local.do"` then `$data_root` | Shared-storage bridge |
| LaTeX table | `esttab ... using "$root/output/tables/table.tex", booktabs label replace` | |
| PNG figure | `graph export "$root/output/figures/fig.png", replace width(2400)` | ~300 DPI @ 8in |
